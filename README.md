# exitTime
Изначальный плагин: https://www.blast.hk/threads/61048/
Идея навеяна этим плагином: https://www.blast.hk/threads/51215/
Дело в том, что у сампа есть свой таймер выхода из игры, который стартует при деинициализации, после сохранения настроек, но перед выгрузкой объектов и прочего. По дефолту этот таймер настроен на 2 секунды (2000мс), а данный плагин позволдяет изменить это время.

**Настройки**

Настройка таймера осуществляется в файле exitTime.ini. Так как шаг таймера в samp имеет 100мс, то разницы между 105, 150, 199 и 200мс не будет, а вот разница между 100 и 200мс будет.

Если указать **0мс**, то дополнительно в игре отключатся все вызовы Sleep, что потенциально может дополнительно сократить время выхода.

Если указать **-1мс**, то будет послан пакет о выходе из игры и игра будет сразу закрыта.

Если указать **-2мс**, то будет послан пакет о выходе из игры и игра будет сразу закрыта принудительно.

Пересобрано и переделано нейросетью под версии **R1, R3-1, R5-2, DL-R1-2**

exitTime.ini появляется при первом запуске игры. По умолчанию стоит 1000мс

Можно поместить \modloader\asi\exitTime.asi или папку scripts
